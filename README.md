# AI-Based-Email-Generator
\
AI Email Generator is a full-stack web application that enables users to generate professional emails from simple text prompts. It features user registration and login, instant email generation, automatic email history management, and the ability to view, copy, or delete previously generated emails. Built with React, Spring Boot, Spring Data JPA, Spring Security, and MySQL, the project follows a layered architecture and is configured with Spring AI and Ollama (Llama 3) for AI-powered email generation.
\
# ✨ Features
\
• AI-Based Email Generation from Text Prompts.\
• Automatic Email History Management.\
• View and Delete Generated Emails.\
• Copy Generated Email to Clipboard.\
• Spring AI & Ollama Ready Configuration.\
